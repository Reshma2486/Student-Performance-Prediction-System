# Student Performance Prediction System

## 📌 Project Description
This project uses Machine Learning to predict students' final exam performance based on academic and behavioral data.

## 🧠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## 📂 Project Structure
- data/ → dataset files
- notebook/ → Jupyter notebooks
- Model/ → trained ML model
- images/ → visuals
